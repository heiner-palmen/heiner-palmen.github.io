Privacy Policy

Last updated: March 2026  ·  App: Heiner Palmen YouTube Uploader

1. Overview

Heiner Palmen YouTube Uploader is a personal automation tool developed and operated solely by Heiner Palmen. It uses the YouTube Data API v3 to upload videos to the author's own YouTube channel. This application is not a commercial product and has no end-users other than its author.

2. Data Collection

This application does not collect, store, log, or transmit any personal data from any individual. No analytics, tracking, cookies, or third-party services are used.

3. OAuth & Google API Access

The application requests OAuth 2.0 authorization from Google solely to upload videos to the author's own YouTube account. The OAuth token is stored locally on the author's own machine and is never shared with any third party. This application's use of data received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

4. Data Sharing

No data of any kind is shared with, sold to, or accessible by any third party.

5. Data Retention

The application stores only an OAuth refresh token on the author's local device. This token can be revoked at any time via Google Account Permissions. No other data is retained.

6. Contact

If you have any questions about this privacy policy, you can reach the author via heiner-palmen.github.io.
